В рамках проекта реализована модель CycleGAN на базе PyTorch для задачи переноса стиля между двумя доменами изображений без использования парных данных. В качестве примера используется датасет Monet2Photo, однако код легко адаптируется и для других наборов данных.

В проекте реализованы загрузка и подготовка данных, архитектуры генераторов и дискриминаторов, функции потерь CycleGAN, цикл обучения модели, сохранение чекпоинтов и визуализация результатов. После обучения модель позволяет преобразовывать изображения из одного домена в другой, сохраняя их основное содержание.
